Título

keydown: Dispara quando qualquer tecla é pressionada. É o evento mais utilizado para capturar teclas pois funciona com todas as teclas, incluindo teclas de função.

keypress: Dispara quando uma tecla que gera um caractere é pressionada (como letras, números e alguns símbolos). Não funciona com teclas de função como Shift, Control ou Enter.

keyup: Dispara quando qualquer tecla é liberada.

Event.Target

Representa o elemento que originalmente disparou o evento. É o elemento que foi clicado, pressionado ou teve alguma outra interação que causou o acionamento do evento. Mesmo que o evento tenha sido propagado para outros elementos, event.target sempre indica o elemento que iniciou a cadeia de eventos.
